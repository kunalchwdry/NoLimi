# 🚀 About

**NoLimi** is an AI-powered desktop assistant built using **Python**.

It enables users to interact with their Windows computer using natural voice commands, making everyday tasks faster and more intuitive.

The long-term vision of NoLimi is to combine **Artificial Intelligence, Desktop Automation, and Cybersecurity** into one intelligent assistant.
